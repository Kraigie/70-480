input:in-range

any input that is in a certain range (5-10)

Try typing a number out of range (less than 5 or higher than 10), to see the styling disappear.

Note: The :in-range selector is not supported in Internet Explorer.

input:out-of-range

any input that is outside a certain range (5-10)

Try typing a number within the given range (between 5 and 10), to see the styling disappear.

Note: The :out-of-range selector is not supported in Internet Explorer.
